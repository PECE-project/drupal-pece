PECE Site Administration

How do I add a logo and change site colors?

  1. While logged-in as an administrator, navigate to [SITE URL]/admin/appearance/settings/pece_scholarly_lite
  2. Under the heading, Logo image settings, uncheck the box to use the default logo.
  3. Choose a file to upload as the logo.
  4. To change site colors, scroll down on this page to the heading "MtT Settings."
  5. Click the "Look'n'Feel" link.
  6. Check the box to use a Custom Color Scheme.
  7. Select a Primary color (header and icons), Secondary color (links), and Tertiary color (color to appear on hover in main navigation).
  8. Click "Save Configuration."

How do I approve user memberships?

  1. While logged-in as an administrator, navigate to [SITE URL]/admin/people.
  2. Select 'edit' next to the user you wish to approve.
  3. Change the user status to 'Active' and select the appropriate User Roles.
  4. Click the 'Save' button at the bottom of the page.

How do I add front page image sliders?

  1. While logged-in as an administrator, navigate to [SITE URL]/node/add/frontpage-image-slideshow
  2. Be sure to upload an image scaled and cropped to 960px by 230px, noting that the bottom 40px will be blocked out by the title link.
  3. In the Image Link field, enter the title of the content you would like the slider to link to and the URL directing to that content.
  4. Click Publish.

How do I edit front page image sliders?

  1. While logged-in as an administrator, navigate to [SITE URL]/admin/dashboard/content.
  2. Under 'Type', filter the content list to 'Slideshow Image', and click Apply.
  3. Select the slideshow you'd like to update.
  4. Click on the Edit tab.
  5. Update the relevant fields, and click Save.

How do I add About text?

  1. While logged-in as an administrator, navigate to [SITE URL]/node/add/pece-about-page.
  2. Enter a title (which won't appear on the page), and in the body, enter the site's about text.
  3. Click Publish.

This will appear when users click on the About link in the main navigation, and a snippet of the text will appear on the front page below the image sliders.

How do I set the Featured Essay?

  1. While logged-in as an administrator, navigate to the home page.
  2. Scroll down to the "Featured Essay" block, and click "Edit Featured Essay."
  3. Select a Featured Essay, in the Featured Essay field, and then click "Save."
